1. ¬øCu√°l es la diferencia entre List y Set?

List permite elementos duplicados y mantiene el orden de inserci√≥n. Set no permite duplicados y no garantiza orden.

// Ejemplo en Java
import java.util.*;

List<String> lista = new ArrayList<>();
lista.add("A");
lista.add("A");

Set<String> conjunto = new HashSet<>();
conjunto.add("A");
conjunto.add("A");

System.out.println(lista);    // [A, A]
System.out.println(conjunto); // [A]

4. Ventajas de HashMap frente a TreeMap

HashMap ofrece mejor rendimiento general en operaciones O(1), pero no mantiene orden. TreeMap mantiene orden natural o por comparador, pero con rendimiento O(log n).

Map<String, Integer> hashMap = new HashMap<>();
Map<String, Integer> treeMap = new TreeMap<>();

hashMap.put("b", 2);
hashMap.put("a", 1);
treeMap.put("b", 2);
treeMap.put("a", 1);

System.out.println(hashMap); // Orden no garantizado
System.out.println(treeMap); // Ordenado por clave
